## üéß How to Use
Once you start the song-classifier, follow these simple steps to classify your songs:

1. **Import your audio files:** 
   Drag and drop your audio files into the interface or use the 'Import' button to select files.

2. **Start the classification process:**
   Click on the 'Classify' button to begin analysis. The application quickly scans the audio features of your files.

3. **View results:**
   The application will display the genre classification for each song. You can listen to the audio while viewing the results.

4. **Save the results:** 
   Optionally, you can save the classification results by clicking on 'Export' to generate a report in CSV format.
